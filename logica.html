Operadores

Operadores Aritméticos

Usados para realizar operações matemáticas básicas.

OperadorDescriçãoExemploResultado
+Adição5 + 38
-Subtração5 - 32
*Multiplicação5 * 315
/Divisão15 / 35
%Módulo (resto)17 % 52
++Incrementox++Aumenta x em 1
--Decrementox--Diminui x em 1

Exemplo de Uso

int a = 10;
int b = 3;

printf("Adição: %d\n", a + b);        // 13
printf("Subtração: %d\n", a - b);    // 7
printf("Multiplicação: %d\n", a * b); // 30
printf("Divisão: %d\n", a / b);       // 3
printf("Módulo: %d\n", a % b);       // 1

Operadores Lógicos

Combinam expressões booleanas.

OperadorDescriçãoExemplo
&&AND lógico (verdadeiro se ambas as expressões forem verdadeiras)a > 0 && b < 10
||OR lógico (verdadeiro se uma das expressões for verdadeira)a == 0 || b == 0

Verdadeiro e Falso em C

Em C, 0 é considerado falso. Qualquer valor diferente de 0 é considerado verdadeiro.

if (x) {
    // executa se x != 0
} else {
    // executa se x == 0
}

Precedência de Operadores

A precedência determina a ordem em que as operações são executadas. Operadores com maior precedência são avaliados primeiro.

PrecedênciaOperadoresAssociatividadeExemplo
1 (Máxima)() [] . ->Esquerda para direita(a + b)
2! ~ ++ -- + - * &Direita para esquerda!a, ++x
3* / %Esquerda para direitaa * b / c
4+ -Esquerda para direitaa + b - c
5< <= > >=Esquerda para direitaa < b
6== !=Esquerda para direitaa == b
7&&Esquerda para direitaa && b
8||Esquerda para direitaa || b
9 (Mínima)= += -= *= /=Direita para esquerdaa = b + c

Exemplos de Precedência

int a = 10, b = 5, c = 2;

// Sem parênteses: * tem precedência sobre +
int resultado = a + b * c;  // 20 (não 30)

// Com parênteses: força ordem diferente
resultado = (a + b) * c;  // 30

// Operadores lógicos: && tem precedência sobre ||
if (a > 5 || b > 10 && c > 1) {
    // Avaliado como: (a > 5) || (b > 10 && c > 1)
}

Dica: Use parênteses para deixar claro a ordem de avaliação, mesmo que não sejam obrigatórios. Isso torna o código mais legível e evita erros!

Exemplos Rápidos

int a = 5;
int b = 0;

if (a > 3 && b == 0) {
    printf("Condição satisfeita\n");
}
