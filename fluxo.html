Estruturas de Controle

If - Else

O if avalia uma expressão; se for verdadeira (diferente de 0), executa o bloco. Caso contrário, o else opcional é executado.

if (idade >= 18) {
    printf("Maior de idade\n");
} else {
    printf("Menor de idade\n");
}

Switch

O switch compara uma expressão inteira ou caractere com constantes case. Use break para evitar queda para o próximo caso (fall-through).

switch (opcao) {
    case 1:
        printf("Opção 1\n");
        break;
    case 2:
        printf("Opção 2\n");
        break;
    default:
        printf("Opção inválida\n");
}

Break - Instruções de Parada

A instrução break encerra imediatamente a execução de um loop (for, while, do-while) ou de um switch.

Break em Switch

Sem break, a execução continua para os próximos case (fall-through), o que geralmente causa comportamento indesejado.

❌ Sem Break (Fall-through)

int dia = 2;
switch (dia) {
    case 1:
        printf("Segunda\n");
    case 2:
        printf("Terça\n");
    case 3:
        printf("Quarta\n");
}
// Saída: Terça Quarta (executou case 2 e 3!)

✅ Com Break (Correto)

int dia = 2;
switch (dia) {
    case 1:
        printf("Segunda\n");
        break;
    case 2:
        printf("Terça\n");
        break;
    case 3:
        printf("Quarta\n");
        break;
}
// Saída: Terça (apenas case 2 executado)

Break em Loops

Interrompe a execução do loop, passando para o código após o loop.

Exemplo: Busca em Vetor

int numeros[] = {2, 5, 8, 3, 9};
int alvo = 8;
int encontrado = 0;

for (int i = 0; i < 5; i++) {
    if (numeros[i] == alvo) {
        printf("Encontrado na posição %d\n", i);
        encontrado = 1;
        break;  // Sai do loop imediatamente
    }
}

if (!encontrado) {
    printf("Número não encontrado\n");
}

Exemplo: Menu com Loop

int opcao;
while (1) {
    printf("\n1. Iniciar\n2. Configurar\n3. Sair\nOpção: ");
    scanf("%d", &opcao);
    
    if (opcao == 3) {
        printf("Encerrando...\n");
        break;  // Sai do while infinito
    }
}

Nota: Use break com moderação. Loops com múltiplas condições de saída podem ficar confusos. Para alternativas, considere usar continue para pular iterações.

Loop For

O laço for tem três partes: inicialização, condição e incremento.

for (int i = 0; i < 10; i++) {
    printf("i = %d\n", i);
}

Observação: a inicialização pode declarar a variável (C99+) ou usar uma já existente.

Exercício

1. Quantas vezes o corpo do loop for (int i = 0; i < 3; i++) será executado?

O loop será executado 3 vezes: i = 0, 1 e 2. Ao final i vale 3 e a condição falha.
