Primeiros Passos

Breve História e Propósito da Linguagem C

A linguagem C foi desenvolvida por Dennis Ritchie nos laboratórios Bell (AT&T) no início dos anos 1970 para implementar o sistema operacional UNIX. C tornou-se popular por sua eficiência, controle de baixo nível (acesso à memória), e portabilidade entre plataformas — sendo usada em sistemas operacionais, compiladores, embarcados, ferramentas e bibliotecas.

Visão geral: C oferece um conjunto compacto de construções de linguagem que permitem escrever código de alto desempenho, com controle explícito sobre alocação de memória, tipos primitivos e ponteiros.

Onde o programa começa

Em C, a execução sempre inicia pela função main(). O sistema operacional chama essa função e a partir dela seu programa controla o fluxo.

Ponto e vírgula (;)

C exige que cada instrução termine em ponto e vírgula. Esquecer o ponto e vírgula causa erro de compilação.

#include <stdio.h>

int main(void) {
    printf("Olá, mundo!\n");
    return 0;
}

Incluir bibliotecas

Use #include <nome> para declarar dependências de bibliotecas padrão. Por exemplo, <stdio.h> fornece printf e scanf.

#include <stdio.h>  // entrada/saída padrão

Biblioteca math.h - Operações Matemáticas

A biblioteca <math.h> fornece funções matemáticas avançadas como raiz quadrada, exponenciação, logaritmo, trigonometria e a constante PI.

Funções principais:

  • sqrt(x) — Raiz quadrada
  • pow(base, exp) — Potência (base elevado a exp)
  • ceil(x) — Arredonda para cima
  • floor(x) — Arredonda para baixo
  • fabs(x) — Valor absoluto
  • sin(x), cos(x), tan(x) — Funções trigonométricas
  • log(x) — Logaritmo natural
  • exp(x) — e elevado a x

Constantes:

  • M_PI — Valor de π (3.14159...)
  • M_E — Valor de e (2.71828...)

Exemplos:

#include <stdio.h>
#include <math.h>

int main(void) {
    double x = 16.0;
    
    // Raiz quadrada
    printf("sqrt(16) = %.2f\n", sqrt(x));
    
    // Potência
    printf("pow(2, 3) = %.2f\n", pow(2, 3));
    
    // Raio do círculo
    double raio = 5.0;
    double area = M_PI * pow(raio, 2);
    printf("Área do círculo (r=5) = %.2f\n", area);
    
    // Trigonometria (em radianos)
    printf("sin(π/2) = %.2f\n", sin(M_PI / 2));
    
    // Valor absoluto
    printf("fabs(-7.5) = %.2f\n", fabs(-7.5));
    
    return 0;
}

Importante: Ao compilar com gcc, adicione a flag -lm para linkar a biblioteca matemática: gcc programa.c -o programa -lm

Exercício rápido: compile um arquivo `hello.c` com gcc hello.c -o hello e execute ./hello (no Windows use o executável gerado).

Exercício

1. Qual é o ponto de entrada de um programa em C?

Em C a função main() é o ponto de entrada. Use return 0; para indicar sucesso.
