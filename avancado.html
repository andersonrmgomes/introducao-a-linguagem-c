Matrizes e Ponteiros

Matrizes (Vetores)

Um vetor (array) armazena vários elementos do mesmo tipo em posições contíguas de memória. O primeiro índice é sempre 0.

int v[10];  // índices 0..9
v[0] = 12;
v[9] = 5;  // último elemento

for (int i = 0; i < 10; i++) {
    printf("v[%d] = %d\n", i, v[i]);
}

Acessar um índice fora do intervalo (por exemplo, v[10]) causa comportamento indefinido — pode ler/escrever memória inválida.

Ponteiros

Um ponteiro é uma variável cujo valor é um endereço de memória. Ponteiros são usados para referenciar e modificar dados indiretamente.

int x = 42;
int *px = &x;   // px aponta para x

printf("x = %d, *px = %d\n", x, *px);

*px = 10;     // altera x indiretamente

Operadores:

  • & — operador 'address-of' (ex.: &x).
  • * — operador 'dereference' (ex.: *px).

Ponteiros mal usados podem causar erros graves (segfaults) ou corrupção de memória. Sempre inicialize ponteiros antes de usar.

Acesso a Arquivos de Texto

Em C, você pode ler e escrever arquivos usando funções de I/O. A biblioteca padrão fornece funções como fopen(), fwrite() e fclose().

Função fopen() - Abrir Arquivo

Abre um arquivo e retorna um ponteiro do tipo FILE*.

FILE *fopen(const char *nome, const char *modo)

Modos de abertura:

  • "r" — Leitura (arquivo deve existir)
  • "w" — Escrita (cria ou sobrescreve arquivo)
  • "a" — Append (adiciona ao final do arquivo)
  • "r+" — Leitura e escrita

Escrevendo em Arquivo

FILE *arquivo = fopen("dados.txt", "w");

if (arquivo == NULL) {
    printf("Erro ao abrir arquivo!\n");
    return 1;
}

fprintf(arquivo, "Nome: João\n");
fprintf(arquivo, "Idade: 25\n");

fclose(arquivo);  // sempre fechar o arquivo

Lendo Arquivo Linha por Linha

FILE *arquivo = fopen("dados.txt", "r");

if (arquivo == NULL) {
    printf("Erro ao abrir arquivo!\n");
    return 1;
}

char linha[100];

// fgets() lê até quebra de linha ou EOF
while (fgets(linha, 100, arquivo) != NULL) {
    printf("%s", linha);  // já inclui \n
}

fclose(arquivo);

Lendo Arquivo Caracter por Caracter

FILE *arquivo = fopen("dados.txt", "r");
int c;

// fgetc() retorna EOF (-1) ao final do arquivo
while ((c = fgetc(arquivo)) != EOF) {
    putchar(c);  // imprime o caracter
}

fclose(arquivo);

Adicionando Dados (Append)

FILE *arquivo = fopen("log.txt", "a");

// Adiciona ao final do arquivo sem apagar conteúdo anterior
fprintf(arquivo, "Entrada de log: operação concluída\n");

fclose(arquivo);

Exemplo Completo: Cópia de Arquivo

// Copia conteúdo de um arquivo para outro
FILE *origem = fopen("origem.txt", "r");
FILE *destino = fopen("copia.txt", "w");

if (origem == NULL || destino == NULL) {
    printf("Erro ao abrir arquivo!\n");
    return 1;
}

int c;
while ((c = fgetc(origem)) != EOF) {
    fputc(c, destino);  // escreve no arquivo destino
}

fclose(origem);
fclose(destino);
printf("Arquivo copiado com sucesso!\n");

Boas práticas:

  • Sempre verifique se fopen() retornou NULL
  • Use fclose() para fechar arquivos quando terminar
  • Trate erros de leitura/escrita apropriadamente
  • Use modos apropriados: não misture leitura em "r" com "w"
