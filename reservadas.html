Palavras Reservadas em C

As palavras reservadas não podem ser usadas como identificadores (nomes de variáveis, funções, typedefs, etc.). Abaixo uma lista comum de palavras reservadas em C (C99/C11):

auto      break     case      char
const     continue  default   do
double    else      enum      extern
float     for       goto      if
inline    int       long      register
restrict  return    short     signed
sizeof    static    struct    switch
typedef   union     unsigned  void
volatile  while     _Bool     _Complex
_Imaginary  _Alignas  _Alignof  _Atomic
_Generic  _Noreturn _Static_assert

Observação: algumas palavras (ex.: inline, restrict, palavras com underline) foram introduzidas em versões posteriores do padrão C. Verifique o padrão alvo do seu compilador.
