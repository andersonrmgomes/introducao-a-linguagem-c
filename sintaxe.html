Sintaxe Básica e Tipos

Tipos de Dados Comuns

Tipos básicos em C e exemplos:

TipoDescriçãoExemplo
charCaractere (mínimo 8 bits)char c = 'A';
intInteiro (tamanho dependente da implementação)int n = 42;
floatNúmero de ponto flutuante (precisão simples)float x = 3.14f;
doublePonto flutuante (precisão dupla)double d = 2.71828;

Entrada e Saída: printf e scanf

printf formata e escreve na saída padrão; scanf lê valores da entrada padrão usando especificadores de formato.

printf("Valor inteiro: %d\n", n);
scanf("%d", &n);

Especificadores de formato comuns

EspecificadorTipoExemplo
%dintprintf("%d", 10);
%ccharprintf("%c", 'A');
%ffloatprintf("%f", 3.14f);
%lfdoubleprintf("%lf", 2.71828);
%sString (array de char)printf("%s", nome);
 
// Leitura segura de um inteiro
int idade;
scanf("%d", &idade);

// Exemplo completo
int main(void) {
    int n = 0;
    printf("Digite um número: ");
    scanf("%d", &n);
    printf("Você digitou: %d\n", n);
    return 0;
}

Exercício

1. Qual especificador do printf é usado para imprimir um int?

Use %d para inteiros; %f para float; %s para strings.
