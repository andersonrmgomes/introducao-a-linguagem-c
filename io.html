Entrada e Saída (I/O)

Visão Geral

Em C, a biblioteca padrão stdio.h fornece funções de entrada e saída como printf, scanf, e fgets. Use printf para formatar saída e scanf ou fgets para ler da entrada padrão.

printf e especificadores

printf usa especificadores de formato para converter valores em texto.

Campo de largura e precisão

Ex.: %6d (largura mínima 6), %.2f (2 casas decimais), %8.3f (largura 8, 3 decimais).

scanf e especificadores de entrada

scanf lê dados da entrada padrão segundo especificadores de formato. Sempre use & para obter o endereço das variáveis (exceto arrays e strings).

%f float Lê um valor de ponto flutuante (precisão simples) scanf("%f", &x);
%lf double Lê um valor de ponto flutuante (dupla precisão) scanf("%lf", &y);
%c char Lê um único caractere (incluindo espaços) scanf("%c", &ch);
%s char[] (string) Lê uma sequência de caracteres até o primeiro espaço scanf("%s", name);

%o int Lê um valor octal scanf("%o", &oct);

Limitando a quantidade de caracteres

Para evitar overflow de buffer, especifique o número máximo de caracteres a ler:

char name[32];
scanf("%31s", name);  // lê no máximo 31 caracteres (reserva 1 para '\0')

char buffer[50];
scanf("%49c", buffer);  // lê no máximo 49 caracteres

Pulando espaços e caracteres

Use espaço em branco no formato para pular espaços, ou use * para ignorar valores:

scanf(" %c", &ch);  // espaço pula whitespace antes de ler %c

scanf("%d %*d %d", &a, &c);  // lê inteiro, ignora inteiro, lê inteiro

Problemas comuns e boas práticas

  • Sempre use & antes de variáveis simples — passa o endereço, não o valor.
  • %s é perigoso — lê até o primeiro espaço e pode causar overflow se o buffer for pequeno. Use %31s (por exemplo) para limitar.
  • Verifique o retorno de scanf — retorna o número de itens lidos com sucesso:
int n;
int result = scanf("%d", &n);
if (result != 1) {
    printf("Erro: entrada inválida\n");
}
  • Cuidado ao misturar scanf e fgets — scanf pode deixar \n no buffer, afetando leituras posteriores.
  • Para ler linhas inteiras prefira fgets ao invés de %s.

Exemplo prático completo

#include <stdio.h>

int main(void) {
    int idade;
    float altura;
    double salario;
    char nome[50];
    char iniciais;

    printf("Digite seu nome (máx 49 caracteres): ");
    scanf("%49s", nome);

    printf("Digite sua inicial: ");
    scanf(" %c", &iniciais);  // espaço pula whitespace

    printf("Digite sua idade: ");
    scanf("%d", &idade);

    printf("Digite sua altura (m): ");
    scanf("%f", &altura);

    printf("Digite seu salário (R$): ");
    scanf("%lf", &salario);

    printf("=== Dados ===\n");
    printf("Nome: %s\n", nome);
    printf("Inicial: %c\n", iniciais);
    printf("Idade: %d anos\n", idade);
    printf("Altura: %.2f m\n", altura);
    printf("Salário: R$ %.2lf\n", salario);

    return 0;
}

fgets lê até um máximo de caracteres ou até encontrar '\n'. É a forma segura de ler linhas (substitui gets, que é inseguro).

// Ler uma linha com fgets e remover o newline
char buf[128];
if (fgets(buf, sizeof(buf), stdin) != NULL) {
    // remover '\n' se presente
    size_t len = strcspn(buf, "\n");
    if (buf[len] == '\n') buf[len] = '\0';
    printf("Linha: %s\n", buf);
}

Para usar strcspn inclua string.h. Outra opção é usar sscanf em uma string lida por fgets para parsear números com validação.

Exemplo completo: misturando I/O

#include <stdio.h>
#include <string.h>

int main(void) {
    char nome[64];
    int idade;

    printf("Digite seu nome: ");
    fgets(nome, sizeof(nome), stdin);
    nome[strcspn(nome, "\n")] = '\0';

    printf("Digite sua idade: ");
    if (fgets(buf, sizeof(buf), stdin) != NULL) {
        if (sscanf(buf, "%d", &idade) != 1) {
            printf("Entrada inválida\n");
            return 1;
        }
    }

    printf("Nome: %s, Idade: %d\n", nome, idade);
    return 0;
}

Resumo — Boas práticas

  • Prefira fgets para leitura de strings; remova o '\n' manualmente.
  • Use especificadores de largura em scanf (ex.: %31s).
  • Valide o retorno de scanf e sscanf.
  • Evite gets (removida do padrão). Use fgets.
