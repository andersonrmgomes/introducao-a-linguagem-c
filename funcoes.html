Funções e Modularização

O que é uma Função?

Funções são blocos de código responsáveis por uma tarefa específica. Elas permitem modularizar o programa, tornando-o mais legível e reutilizável.

Sintaxe: tipo_retorno nome_funcao(parâmetros) { corpo }

Funções com Retorno

Funções que retornam um valor ao código que as chamou. Use return para enviar o valor.

Exemplo 1: Função Simples

// Função que soma dois inteiros e retorna o resultado
int soma(int a, int b) {
    return a + b;
}

int main(void) {
    int r = soma(2, 3);  // r = 5
    printf("Resultado = %d\n", r);
    return 0;
}

Exemplo 2: Função com Mais Lógica

// Função que calcula a média de três notas
float media(float n1, float n2, float n3) {
    float resultado = (n1 + n2 + n3) / 3.0;
    return resultado;
}

int main(void) {
    float m = media(7.5, 8.0, 6.5);
    printf("Média = %.2f\n", m);  // 7.33
    return 0;
}

Funções Sem Retorno (void)

Funções que não retornam valor, apenas executam uma ação. Use o tipo void como tipo de retorno.

Exemplo 1: Função que Imprime

// Função que imprime uma mensagem (sem retorno)
void saudacao(char *nome) {
    printf("Olá, %s!\n", nome);
    // Não há 'return' pois é void
}

int main(void) {
    saudacao("João");    // Olá, João!
    saudacao("Maria");   // Olá, Maria!
    return 0;
}

Exemplo 2: Função que Modifica Dados

// Função que modifica valores através de ponteiros
void incrementar(int *valor) {
    (*valor)++;  // incrementa o valor apontado
}

int main(void) {
    int x = 5;
    printf("Antes: %d\n", x);      // 5
    incrementar(&x);
    printf("Depois: %d\n", x);     // 6
    return 0;
}

Vetores (Matrizes Unidimensionais)

Vetores armazenam múltiplos elementos do mesmo tipo em posições de memória contíguas. Em C, o primeiro índice é 0.

Declaração e Inicialização

// Declaração sem inicialização
int notas[5];  // cria vetor com 5 posições (índices 0..4)

// Inicialização ao declarar
int valores[] = {10, 20, 30, 40, 50};
char vogais[] = {'a', 'e', 'i', 'o', 'u'};

// Acessar elementos
notas[0] = 8;      // primeiro elemento
notas[4] = 9;      // último elemento (índice = tamanho - 1)

Iterando sobre Vetores

int numeros[] = {10, 20, 30, 40};
int tamanho = 4;

// Imprimindo todos os elementos
for (int i = 0; i < tamanho; i++) {
    printf("%d\n", numeros[i]);
}

Funções com Vetores

// Função que soma todos os elementos de um vetor
int somar_vetor(int *v, int tamanho) {
    int soma = 0;
    for (int i = 0; i < tamanho; i++) {
        soma += v[i];
    }
    return soma;
}

int main(void) {
    int valores[] = {1, 2, 3, 4, 5};
    int resultado = somar_vetor(valores, 5);
    printf("Soma = %d\n", resultado);  // 15
    return 0;
}

Importante: Ao passar um vetor para uma função, passa-se um ponteiro para o primeiro elemento. Por isso use int *v ou int v[] (equivalentes). Sempre passe o tamanho do vetor como parâmetro!

Matrizes Bidimensionais

Matrizes com linhas e colunas. Ideais para representar dados em grade.

// Declaração de matriz 3x3
int matriz[3][3];

// Inicialização
int grid[2][2] = {
    {1, 2},
    {3, 4}
};

// Acessando elementos
matriz[0][0] = 5;  // linha 0, coluna 0

// Iterando em matriz
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        printf("%d ", matriz[i][j]);
    }
    printf("\n");
}
